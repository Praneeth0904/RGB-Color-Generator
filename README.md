## 📌 RGB-Color-Generator

Generates random RGB color values and displays them dynamically with real-time UI updates.


---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Generates random RGB color values dynamically.
* Updates the background color in real-time.
* Helps understand color formats and JavaScript randomness.
